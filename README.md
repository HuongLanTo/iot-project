1. Install package

  ```shell
  cd server OR cd web_client 
  npm i
  ```

2. Run server

  ```shell
  cd server
  npm start
  ```
  
3. Run web client

  ```shell
  cd web_client
  ng serve
  ```
  
4. Run format all file
  
  ```shell
  cd server OR cd web_client
  npm run pretty
  ```
